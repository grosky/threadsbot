🧵 Lazy Threads

AI assistant for Threads content creators, delivered as a Telegram bot.

What it does

  • Generates Threads posts in 5 viral formats using Google Gemini
  • Turns voice notes into polished storytelling posts
  • Analyses Threads profile screenshots for packaging improvements
  • Publishes directly to Threads via the official Threads API (OAuth)
  • Breaks long posts into native Threads chains automatically

How it uses Threads API

  • threads_basic — to identify the authenticated user (id, username) after OAuth
  • threads_content_publish — to create and publish posts on the user's behalf, only when explicitly triggered by the user clicking «Publish» in the bot
